Web Development

Course Overview

Lecturer : Daniel Dang

This course provided a comprehensive introduction to web development, focusing on the core building blocks of modern web design and development. Using a mobile-first approach, I learned to design, plan, and build fully functional websites that are responsive and user-friendly.

Core Technologies Learned

HTML (HyperText Markup Language)

Mastered semantic HTML5 structure for creating well-organised, accessible web pages. Learned proper use of elements, forms, and document structure.

CSS (Cascading Style Sheets)

Developed proficiency in styling web pages, including layout techniques, responsive design, flexbox, and creating visually appealing user interfaces.

JavaScript

Learned JavaScript fundamentals for adding interactivity and dynamic behavior to web pages, enhancing user experience.

Mobile-First Approach

Adopted a mobile-first design philosophy, ensuring websites are optimised for mobile devices first, then enhanced for larger screens.

Design and Development Process

Website Planning and Design

Learned the importance of planning before coding. Developed skills in understanding client requirements and translating them into technical specifications.

Wireframing

Created wireframes to visualise website layout and functionality before implementation. This process helped ensure clear communication and efficient development.
This also provided the building blocks for the creation and implementation of the final website.

Functionality Discussion

Engaged in detailed discussions about website functionality, user experience, and technical requirements to ensure all stakeholders understood the final product.
Covered Accessibilty Guidelines and implementation.

Full Website Implementation

Built fully operational websites from scratch, applying all learned concepts. Implemented responsive design, proper HTML structure, and CSS styling to create professional, functional websites.

Key Learning Outcomes

  • Ability to create semantic, accessible HTML documents
  • Proficiency in CSS for styling and responsive design
  • Understanding of JavaScript for adding interactivity
  • Mobile-first design and responsive web development
  • Web Accessibility, design, implementation and testing
  • Web design planning and wireframing skills
  • Full-stack website development from concept to deployment
  • Best practices in web development and user experience
  • Copyright, Privacy and Ethical practice

Skills Developed

Through this course, I have developed the ability to create modern, responsive websites that work seamlessly across all devices.
I understand the importance of planning and design before implementation, and I can effectively translate design concepts into functional, professional websites.
This foundation prepares me for more advanced web development frameworks and technologies.
