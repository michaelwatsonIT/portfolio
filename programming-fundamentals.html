Programming Fundamentals

Course Overview

Lecturer : John Jamieson

This course introduced me to programming using Python, one of the most versatile and beginner-friendly programming languages. Through a series of progressively complex assignments, I developed a strong foundation in programming logic, data structures, and object-oriented programming principles.

Programming Language: Python

Python was chosen as the primary language for this course due to its readability, simplicity, and widespread use in industry. Python as a foundational programming language allows for learning and development that connects to many other languages in industry. Throughout the semester, I progressed from basic syntax and logic to advanced concepts like object-oriented programming and graphical user interfaces.

Assignments and Projects

E-Reader Program

Created a simple e-reader program capable of reading and displaying 10 different books, in basic UTF-8 text format. This assignment introduced me to file handling, data structures, loops, encoding and fuctions. I learned how to organise data, manage file operations, and create a functional application that could handle multiple books and user input and interactions.

Grade Calculator with GUI

Developed a grade calculator application using the guizero library to create a graphical user interface. This project introduced me to GUI programming and event-driven development. I learned how to create interactive interfaces, handle user input, and perform calculations based on that input. The application demonstrated the practical application of programming concepts in a user-friendly format.

Cinema Administration Program

The final and most complex assignment required creating a Cinema Administration Program using object-oriented programming principles. This project involved designing and implementing classes, methods, and refactoring a provided GUI. Key accomplishments included:

  • Designing and implementing multiple classes to represent cinema entities (customers, movies, and bookings)
  • Creating methods for managing cinema operations (build customer profiles, adding movies, managing bookings)
  • Applying object-oriented principles such as encapsulation, inheritance, and polymorphism
  • Refactoring and customising a provided GUI to meet project requirements
  • Integrating the GUI with the backend logic to create a fully functional application

E-Reader

Key Concepts

  • Python syntax and basic programming logic
  • Data types, variables, and operators
  • Control structures (if statements, loops)
  • Functions and code organisation
  • File handling and data persistence
  • Object-oriented programming (classes, methods, attributes)
  • Encapsulation and code organisation
  • GUI development with guizero
  • Event-driven programming
  • Code refactoring and optimisation
  • Standard practices and documentation

Skills Developed

Through this course, I have developed a solid foundation in programming that extends beyond Python. I understand fundamental programming concepts that apply to any language, including logic, data structures, and object-oriented design. I can write clean, organised code, design applications with proper structure, and create user-friendly interfaces. These skills provide a strong foundation for learning additional programming languages and frameworks in the future.
